# Monitor
  Мониторинг состояния Linux-системы.
  
# Требования
  Linux OS
  
  C++17
  
  CMake 3.10 или выше
  
  Библиотека nlohmann/json

# Установка
  Клонируйте репозиторий:
  
  git clone https://github.com/Korelin013/Monitor
  
  cd Monitor

# Создайте директорию для сборки и выполните сборку:
  mkdir build
  
  cd build
  
  cmake ..
  
  make

# Запуск
  ./Monitor

# Пример файла конфигурации
  config.json

# Пример лог файла
  file.log
